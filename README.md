# ERP Demo

Application de gestion (CRM/ERP) construite avec Blazor Server, MudBlazor et Entity Framework Core.

## Stack technique
- ASP.NET 10 Core / Blazor Server
- MudBlazor
- Entity Framework Core + SQL Server
- Architecture MVVM

## Fonctionnalités
- Authentification avec rôles
- Gestion des commandes et produits
- DataGrid Générique côté développeur (Filtres, Pagination, Tri, Vues)
- Export Excel/CSV/PDF
- Dashboard avec charts
- Seed de données de test automatique
